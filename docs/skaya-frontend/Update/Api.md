- Add a custom [Design and Layout](https://docusaurus.io/docs/styling-layout)
- Add a [search bar](https://docusaurus.io/docs/search)
- Find inspirations in the [Skaya showcase](https://docusaurus.io/showcase)
- Get involved in the [Skaya Community](https://docusaurus.io/community/support)
